This plugin provides Widget for Zen service

== Description ==
Widget for Zen plugin allows you to add widget on the sidebar, that will display
preview of the last post published on your Zen blog.

To configure plugin please check "Settings/Widget for Zen" page in your admin console.
If you like the plugin please give five start for it.

== Frequently Asked Questions ==
= Installation =
1. Install plugin as any other using your admin console. "Plugins / Add New".
2. Activate plugin in Plugins Menu.
3. Configure plugins (Settings / Widget for Zen)
4. Place widget on your sidebar and enjoy!

= Why I need this plugin? =
If you have a Zen Channel and want to increase number of readers
and subscribers of the channel you will find this plugin really useful.
It will create a preview of you Zen Posts and only one click will be needed to read your Zen posts.
It's a most usable way to engage more readers to your channel.

= Where I can see how plugin works? =
You can check my [site](https://prozen.ru). Links on left sidebar generated by the plugin.
